Atmosfera

An immersive weather experience built with HTML, CSS, JavaScript and the OpenWeatherMap API.

OVERVIEW

What is Atmosfera?

Atmosfera is a cinematic weather web application that transforms how you experience weather data. Instead of a static UI, the entire visual world — background gradient, particle system, 3D orb colours, and lighting — shifts dynamically when you search a city. Search a rainy city and the background turns deep blue with animated rain particles. Search a sunny city and the world turns warm amber with floating light motes.

Built as a frontend internship project, it demonstrates real API integration using JavaScript's fetch() and async/await, canvas-based particle systems, 3D CSS transforms, and dynamic theming via CSS variables.

API INTEGRATION

How Weather Data is Fetched

The app uses the OpenWeatherMap REST API. When you search a city, JavaScript sends an HTTP GET request to OpenWeatherMap's servers. The response comes back as a JSON object containing all weather data, which is then parsed and injected into the DOM.

async function fetchWeather(city) {
  const url = `https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${API_KEY}`;
  const response = await fetch(url); // send HTTP request
  const data = await response.json(); // parse JSON
  renderWeather(data); // inject into DOM
  applyWorld(data.weather[0].main); // change theme
}
