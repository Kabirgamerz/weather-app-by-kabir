About SkyCast

A real-time weather application built with HTML, CSS, and JavaScript using the OpenWeatherMap API.

Home/About

🌤️ What is SkyCast?

SkyCast is a browser-based weather application that fetches real-time weather data from the OpenWeatherMap API. Users can search any city in the world to get current conditions — temperature, humidity, wind speed, pressure, visibility, sunrise, and sunset times.

The app also features a 5-day forecast with an hourly breakdown, automatic background themes that change based on weather conditions, geolocation detection, and a °C/°F toggle — all built with vanilla JavaScript and no frameworks.

⚡ How Weather Data is Fetched from an API

This application uses the OpenWeatherMap REST API to retrieve live weather data. Here's exactly how the process works step by step:

👤 User types city
→
⚡ JS calls fetch()
→
🌐 HTTP request sent
→
📡 API server responds
→
📦 JSON parsed by JS
→
🖥️ Data shown on screen

The JavaScript code uses async/await to make HTTP requests. The fetch() function is built into modern browsers and returns a Promise. The API sends back a JSON object which is then parsed and rendered into the page HTML dynamically using DOM manipulation.

// How the API call works in script.js
async function fetchWeather(city) {
  const url = `https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${API_KEY}`;
  const response = await fetch(url);  // send request
  const data = await response.json(); // parse JSON
  renderWeather(data);              // show on page
}

🚀 Features

🔍
City Search Search any city in the world for real-time weather conditions.
📅
5-Day Forecast Extended forecast with daily high/low temperatures and conditions, plus an hourly strip.
📍
Geolocation Detect your current location automatically using the browser's Geolocation API.
🎨
Dynamic Backgrounds The app's gradient background changes based on the weather — sunny, rainy, stormy, snowy, and more.
🌡️
°C / °F Toggle Switch between Celsius and Fahrenheit with one click. The data re-fetches with the correct unit.
⚠️
Error Handling Invalid city names and network failures show user-friendly error messages with guidance to try again.

🛠️ Technology Used

🌐
HTML5 Semantic structure with header, nav, main, section, article, and footer elements. ARIA labels for accessibility.
🎨
CSS3 CSS Variables, glassmorphism effects with backdrop-filter, dynamic gradient themes, Flexbox/Grid layout, and responsive media queries.
⚡
JavaScript (ES6+) async/await with fetch(), DOM manipulation, Geolocation API, error handling, and live clock using setInterval.
📡
OpenWeatherMap API Free REST API providing current weather and 5-day forecast data for any city. Returns data in JSON format.
